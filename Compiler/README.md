### Projet de compilateur pour processeur 8 bit


Operation arithmetique nessessaire : 

- Addition 

- Soustraction

- ET binaire

- OU binaire

- NOT binaire

- >> décalage a droite

- << décalage a gauche

Type de donnée nessessaire : 

- 8 bits signée 

- 8 bits non signée

Input - output : 

- Bus de donnée 4 bit


-0-00000-00-

-taille de l'instruction - upcode - arg 1 - 

Listes des commandes nessessaires : 

SET
ADD //signé et non signé
SUB //non signé
AND
OR
NOT
LSHF
RSHF

SAVE
